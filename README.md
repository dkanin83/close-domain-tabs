# Close Domain Tabs

Расширение для Chrome/Edge для закрытия всех вкладок определенного домена.

## Возможности

- 📂 Группировка открытых вкладок по доменам
- 🏠 Закрытие только основного домена
- 🌿 Закрытие только поддоменов
- 🗑️ Закрытие домена со всеми поддоменами
- 📜 История закрытых доменов
- 🔄 Обновление списка вкладок
- 📌 Опция закрытия закрепленных вкладок
- 🔗 Переход на вкладку по клику на домен

## Установка

### Из исходников
1. Скачайте репозиторий
2. Откройте `chrome://extensions/`
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с расширением

### Из .crx файла
1. Перетащите `.crx` файл в `chrome://extensions/`
2. Подтвердите установку

## Использование

1. Нажмите на иконку расширения
2. Введите домен в поле ввода
3. Нажмите "Закрыть все вкладки домена"
4. Или выберите домен из списка открытых вкладок
